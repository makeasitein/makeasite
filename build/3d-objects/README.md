# 3D Objects from Freepik

This directory contains 3D objects downloaded from Freepik.

## How to Add 3D Objects

1. Visit [Freepik 3D Models](https://www.freepik.com/3d-models)
2. Search for relevant 3D objects like:
   - Computer/laptop 3D models for tech sections
   - Business icons and symbols
   - Abstract shapes and geometric objects
   - People/characters for testimonials
   - Product mockups

3. Download PNG renders of 3D objects (with transparent backgrounds when possible)
4. Place them in this directory
5. Reference them in your components

## Recommended 3D Assets for this Website:

- **laptop-3d.png** - Modern laptop for hero section
- **chart-3d.png** - 3D chart/graph for services
- **rocket-3d.png** - 3D rocket for growth/success
- **target-3d.png** - 3D target for goals/precision
- **shield-3d.png** - 3D shield for security/trust
- **globe-3d.png** - 3D globe for global reach
- **abstract-shape-1.png** - Abstract floating shape
- **abstract-shape-2.png** - Abstract floating shape
- **abstract-shape-3.png** - Abstract floating shape

## Usage Example:

```tsx
<img 
  src="/3d-objects/laptop-3d.png"
```
